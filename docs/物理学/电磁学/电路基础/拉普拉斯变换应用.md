# 拉普拉斯变换应用

## 电路元件模型

### 电阻

对于电阻，根据欧姆定律，有：

$$
v(t) = R i(t)
$$

对上式左右两边进行拉普拉斯变换，有：

$$
\boxed{
V(s) = R I(s)
}
$$

### 电感

对于电感，根据法拉第电磁感应定律，有：

$$
v(t) = L \frac{\mathrm{d}i(t)}{\mathrm{d}t}
$$

对上式左右两边进行拉普拉斯变换，有：

$$
V(s) = L \left[ s I(s) - i(0^-) \right] = s L I(s) - L i(0^-)
$$

或：

$$
\boxed{
I(s) = \frac{1}{s L} V(s) + \frac{i(0^-)}{s}
}
$$

### 电容

对于电容，根据电容的定义，有：

$$
i(t) = C \frac{\mathrm{d}v(t)}{\mathrm{d}t}
$$

对上式左右两边进行拉普拉斯变换，有：

$$
I(s) = C \left[ s V(s) - v(0^-) \right] = s C V(s) - C v(0^-)
$$

或：

$$
\boxed{
V(s) = \frac{1}{s C} I(s) + \frac{v(0^-)}{s}
}
$$

## 传递函数

The **transfer function** H(s) is the ratio of the output response Y(s) to the input excitation X(s), assuming all initial conditions are zero.

$$
\boxed{
H(s) = \frac{Y(s)}{X(s)}
}
$$

传递函数取决于如何定义输入和输出。由于输入和输出可以是电路中任何位置的电流或电压，因此有四种可能的传递函数：

$$
\begin{aligned}
H(s) = \text{Voltage gain} = \frac{V_o(s)}{V_i(s)}\\[8pt]
H(s) = \text{Current gain} = \frac{I_o(s)}{I_i(s)}\\[8pt]
H(s) = \text{Impedance} = \frac{V(s)}{I(s)}\\[8pt]
H(s) = \text{Admittance} = \frac{I(s)}{V(s)}
\end{aligned}
$$

## 状态变量

$$
\boxed{
\begin{aligned}
\mathbf{\dot{x}(t)} = \mathbf{A} \mathbf{x}(t) + \mathbf{B} \mathbf{z}(t)\\[8pt]
\mathbf{y}(t) = \mathbf{C} \mathbf{x}(t) + \mathbf{D} \mathbf{z}(t)
\end{aligned}
}
\begin{aligned}
\quad (1) \\[8pt]
\quad (2)
\end{aligned}
$$

假设初始条件为零，对公式 (1) 进行拉普拉斯变换，有：

$$
\begin{aligned}
s \mathbf{X}(s) &= \mathbf{A} \mathbf{X}(s) + \mathbf{B} \mathbf{Z}(s)\\[8pt]
(s \mathbf{I} - \mathbf{A}) \mathbf{X}(s) &= \mathbf{B} \mathbf{Z}(s)\\[8pt]
\mathbf{X}(s) &= (s \mathbf{I} - \mathbf{A})^{-1} \mathbf{B} \mathbf{Z}(s)
\end{aligned}
$$

对公式 (2) 进行拉普拉斯变换，有：

$$
\mathbf{Y}(s) = \mathbf{C} \mathbf{X}(s) + \mathbf{D} \mathbf{Z}(s)
$$

于是，传递函数为：

$$
\begin{aligned}
\mathbf{H}(s) 
&= \frac{\mathbf{Y}(s)}{\mathbf{Z}(s)}\\[8pt]
&= \frac{\mathbf{C} (s \mathbf{I} - \mathbf{A})^{-1} \mathbf{B} \mathbf{Z}(s) + \mathbf{D} \mathbf{Z}(s)}{\mathbf{Z}(s)}\\[8pt]
&= \mathbf{C} (s \mathbf{I} - \mathbf{A})^{-1} \mathbf{B} + \mathbf{D}
\end{aligned}
$$

在大多数情况下，$\mathbf{D}=0$，于是有：

$$
\boxed{
\mathbf{H}(s) = \mathbf{C} (s \mathbf{I} - \mathbf{A})^{-1} \mathbf{B}
}
$$
