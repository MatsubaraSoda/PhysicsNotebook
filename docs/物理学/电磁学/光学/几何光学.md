# 几何光学

Geometrical Optics

## 几何光学近似

Geometrical Optics Approximation

大自然的底层代码是**波动光学 (Wave Optics)**。任何现实中 (realizable) 的镜头由于尺寸有限，必然会切断光波，产生连续荡漾的**衍射 (diffraction)** 现象，因此绝对完美的“点对点成像”是不存在的，永远会留下**模糊光斑 (blur spot)**。

所谓的**几何光学 (Geometrical Optics)**，并不是大自然的真相，而是人类工程师做的一场极致的“妥协与整容”。

在数学上，当波长相对透镜尺寸极小，逼近极限 ($\lambda_0 \to 0$) 时，衍射效应缩减到微乎其微 (no longer observable)。此时，我们**客观上忽略 (neglect)** 波动性带来的微小误差，假装光沿**绝对直线 (rectilinear)** 传播。

我们之所以热爱这个**理论上虚构的领域 (conceptual domain)**，是因为它扔掉了复杂的微分方程，带来了用直尺就能绘制光路的**巨大便利 (simplicity)**。

## 非球面

### 圆锥曲线

先简单回顾一下圆锥曲线中圆、椭圆、抛物线和双曲线的定义。

- **圆 Circle**

给定圆心 $(x_0, y_0)$，半径 $r$，则圆为

$$
\left\{ (x,y) \in \mathbb{R}^2 \mid \sqrt{x^2 + y^2} = r \right\}
$$

- **椭圆 Ellipse**

给定两个焦点 $F_1(-c, 0)$ 和 $F_2(c, 0)$，以及一个常数 $2a$ （满足 $a > c > 0$），则椭圆轨迹为

$$
\left\{ (x,y) \in \mathbb{R}^2 \mid \sqrt{(x+c)^2 + y^2} + \sqrt{(x-c)^2 + y^2} = 2a \right\}
$$

> 这种定义符合双焦点绳子作椭圆思维，具有直观性。两个焦点处放置木桩，将一根长 $2a$ 的绳子两头系在木桩上，拉住并使其绷紧，旋转一周便能够得到椭圆轨道。

<br/>

给定一个焦点 $F(c, 0)$，一条准线 $\displaystyle x = \frac{a^2}{c}$，以及离心率 $\displaystyle e=\frac{c}{a}$ （满足 $0 < e < 1$，亦为 $a > c > 0$）。则椭圆轨迹为

$$
\left\{ (x,y) \in \mathbb{R}^2 \mid \frac{\sqrt{(x-c)^2+y^2}}{\left| x-\frac{a^2}{c} \right|} = e \right\}
$$

> 椭圆轨道上的点到焦点的距离与到准线的距离之比等于离心率 $e$。

<br/>

通常我们更多使用以下标准方程形式（其中 $b^2 = a^2 - c^2$）

$$
\left\{ (x,y) \in \mathbb{R}^2 \mid \frac{x^2}{a^2} + \frac{y^2}{b^2} =1 \right\}
$$

- 抛物线 Parabola

给定一个焦点 $F(\frac{p}{2}, 0)$，一条准线 $\displaystyle x = -\frac{p}{2}$，以及离心率 $e=1$ （满足 $p > 0$）。则抛物线轨迹为

$$
\left\{ (x,y) \in \mathbb{R}^2 \mid \frac{\sqrt{(x-\frac{p}{2})^2+y^2}}{\left| x+\frac{p}{2} \right|} = 1 \right\}
$$

> 抛物线轨迹上的点到焦点的距离与到准线的距离严格相等。它可以被视为椭圆在 $e \to 1$ 时，另一端在无穷远处拉断并炸开的极限退化形态，

<br/>

通常我们更多使用以下标准方程形式（以顶点在原点，开口向右为例）

$$
\left\{ (x,y) \in \mathbb{R}^2 \mid y^2 = 2px \right\}
$$

- 双曲线 Hyperbola

给定两个焦点 $F_1(-c, 0)$ 和 $F_2(c, 0)$，以及一个常数 $2a$ （满足 $c > a > 0$），则双曲线轨迹为

$$\left\{ (x,y) \in \mathbb{R}^2 \mid \left| \sqrt{(x+c)^2 + y^2} - \sqrt{(x-c)^2 + y^2} \right| = 2a \right\}$$

> 这种定义描述了动点到两焦点距离之差的绝对值为定值。与椭圆的“距离和恒定”相对应，双曲线体现的是空间中的“距离差恒定”（在物理上可以对应干涉条纹或引力等势差）。

<br/>

给定一个焦点 $F(c, 0)$，一条准线 $\displaystyle x = \frac{a^2}{c}$，以及离心率 $\displaystyle e=\frac{c}{a}$ （满足 $e > 1$，亦为 $c > a > 0$）。则双曲线轨迹为

$$\left\{ (x,y) \in \mathbb{R}^2 \mid \frac{\sqrt{(x-c)^2+y^2}}{\left| x-\frac{a^2}{c} \right|} = e \right\}$$

> 双曲线轨迹上的点到焦点的距离与到准线的距离之比等于离心率 $e$。它在代数形式上与椭圆的焦准比定义一模一样，仅仅是离心率 $e$ 跨过了 1 的界限，造就了向外张开的图形。

<br/>

通常我们更多使用以下标准方程形式（其中 $b^2 = c^2 - a^2$）

$$\left\{ (x,y) \in \mathbb{R}^2 \mid \frac{x^2}{a^2} - \frac{y^2}{b^2} = 1 \right\}$$
