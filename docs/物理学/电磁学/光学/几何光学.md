# 几何光学

Geometrical Optics

## 几何光学近似

Geometrical Optics Approximation

大自然的底层代码是**波动光学 (Wave Optics)**。任何现实中 (realizable) 的镜头由于尺寸有限，必然会切断光波，产生连续荡漾的**衍射 (diffraction)** 现象，因此绝对完美的“点对点成像”是不存在的，永远会留下**模糊光斑 (blur spot)**。

所谓的**几何光学 (Geometrical Optics)**，并不是大自然的真相，而是人类工程师做的一场极致的“妥协与整容”。

在数学上，当波长相对透镜尺寸极小，逼近极限 ($\lambda_0 \to 0$) 时，衍射效应缩减到微乎其微 (no longer observable)。此时，我们**客观上忽略 (neglect)** 波动性带来的微小误差，假装光沿**绝对直线 (rectilinear)** 传播。

我们之所以热爱这个**理论上虚构的领域 (conceptual domain)**，是因为它扔掉了复杂的微分方程，带来了用直尺就能绘制光路的**巨大便利 (simplicity)**。
